Introduction

This is an introduction to the language and environment.

It covers why it is useful and where it can be applied.

Getting started is easy with proper guidance.

Make sure to install the latest version.

Community support is robust and active.

Setup

Install using a package manager:

npm install tool

Set environment variables:

export PATH=$PATH:/path/to/bin

Verify installation:

tool --version

Create your first project:

tool init

Open your editor and start coding!

Functions

Functions help you reuse code.

They can take parameters and return values.

function greet(name) { return 'Hello ' + name; }

Functions can be anonymous or named.

Arrow functions are a shorter syntax.
